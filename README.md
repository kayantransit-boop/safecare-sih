# 🏥 Kayan Clinique — Système d'Information Hospitalier

Logiciel de gestion hospitalière complet (SIH) développé pour **Kayan Clinique**.

---

## Modules

| Module | Fonctionnalités |
|--------|----------------|
| 👥 **Patients** | Dossiers patients, signes vitaux, score NEWS2 automatique |
| 🛏️ **Séjours** | Admissions, hospitalisations, sorties |
| 🔬 **Laboratoire** | Prescriptions d'analyses, résultats, alertes critiques |
| 🧾 **Facturation** | Factures, paiements, export PDF |
| 👨‍⚕️ **Personnel** | Gestion de l'équipe médicale |

## Technologies

- **Frontend** : React 18 + Vite
- **Backend** : Node.js + Fastify + TypeScript
- **Base de données** : PostgreSQL
- **PDF** : jsPDF + jsPDF-AutoTable

## Démarrage local

```bash
# Installer les dépendances
INSTALLER.bat

# Lancer l'application
LANCER.bat
```

- API : http://localhost:3001
- Application : http://localhost:3000
- Connexion : `admin` / `admin123`
